# Commands

- `npm run dev` — start the Next.js dev server (http://localhost:3000)
- `npm run build` — production build; run before claiming a task is done if you touched routing, config, or types
- `npm run lint` — ESLint (flat config, `eslint-config-next` + `eslint-config-prettier`). Must pass with zero warnings.

There is no test runner configured. Do not invent `npm test`.
