# shadcn/ui primitives

These files are generated by the shadcn CLI under `style: "radix-nova"` (see `components.json`). Treat them as semi-vendored.

- Prefer regenerating via `npx shadcn@latest add <name>` over hand-editing. The generator wires up the correct Radix primitives, CVA variants, and class conventions for this project's style.
- If a real edit is unavoidable, only touch styling/variants. Don't rewire the underlying Radix primitive — replace the whole file with the shadcn CLI instead.
- Icon imports should come from `lucide-react` (matches `iconLibrary` in `components.json`).
